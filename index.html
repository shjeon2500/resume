성장 과정

저는 경험이 자산이라 생각하며 성장 해 왔습니다. 대학 전공에서의 처음 배우는 과목들도 계속 된 시도를 거듭하여 점점 이해와 슥듭이 빨라지는 자신을 느끼며 전공과목에 대한 호기심을 갖게 되었습니다. 실습을 통해 먼저 몸으로 익히면서 배운 것들이 좀더 오래 남는것을 몸소 느꼈고, 그 후에도 새로운 것들을 몸으로 먼저 익히는 습관을 가지기 위해 노력해왔습니다.

또한 대학생활중 수업 내용이나 조별 과제에서 역량 부족으로 힘들어 하던 대학동기들을 위해 남들보다 더 공부해가면서 어떻게 동기들에게 조금 더 쉽게 설명할 수 있는지 연구하였고, 동기들에게 개별적으로 과외를 해주던게 한 학기가 지날 때 쯤에는 빈 강의실에서 약 10명 정도의 작지만 강의도 하였습니다. 이를 통해 동기들과의 화합과 제 자신의 역량도 더 늘리는 계기가 되었습니다.

성격의 장 / 단점

제 성격의 가장 큰 장점은 호기심과 배려심이 많은 것입니다. 대학 졸업 후에도 계속 전공에 대한 과목들을 좀 더 알아가고 싶은 마음과 그것을 알게 되었을 때를 가장 즐깁니다. 또한 친구들이나 대학 동기들 등등 여러 모임 장소에서도 상대방의 기분을 잘 파악하고 때때로 개인적인 상담을 받아주는 역할도 하는것이 저의 가장 큰 장점입니다.

단점은 다소 느긋한 성격인 점입니다. 이를 극복하기 위해 메모하는 습관과 미리미리 준비하는 습관을 기르기 위해 노력하였습니다.

배워왔던 것들

3년제 컴퓨터 전자과에서 처음으로 C/C++을 접해 기초 언어를 배우고 3년동안 AUTO CAD, MATLAB, cadence orcad, LABVIEW 등등 여러 프로그램 툴을 다뤄 보았습니다.

3년제를 졸업 후 전기공학과에 편입하여 심화과정으로 1년간 자동제어(승강기)를 배우며 승강기 회사에서 계약직으로 1년간 사회 생활을 몸으로 익히면서 계약직 만료와 동시에 C언어의 지식을 바탕으로 Java를 공부하였습니다. Java로는 swing을 이용한 Scroll Runnnig Action Game을 만들 보았습니다.

Java 이후로는 Html 와 CSS 를 배워 bootstrap을 이용하여 현재 이력서를 작성하고 있습니다. 그리고 JSP/Servlet 을 이용한 자유게시판, 회원간의 1:1 채팅, 강의 평가 등록 사이트를 만들었습니다.

현재는 Java, Html, CSS, JSP 및 Servlet을 이용하여 1인 프로젝트로 영화 관련 사이트를 만들고 있습니다.

Scroll Runnnig Action Game

2018 평창동계올림픽을 기념하기 위하여 평창동계올림픽의 마스코트 캐릭터 ‘수호랑’을 활용하여 Scroll Running Action Gmae의 기능을 입힌 쉽고 재미있는 홍보용 게임을 개발 하고자 하였습니다.

오직 Java만 설계를 하였습니다. Scroll Running Action Gmae의 가장 기본적인 기능인 횡이동과 키 입력시 점프와 2단 점프 및 슬라이드를 하는 모션을 취하고 장애물에 충돌시 체력게이지가 큰 폭으로 줄어들면서 캐릭터는 약 1초 가량의 무적 상태가 됩니다. 일시정지를 누르면 게임 화면 정지와 나가기 버튼으로 게임 중간에 바로 점수화면으로 이동하고 랭킹버튼은 지금까지의 점수중 상위 3등까지의 점수를 보여줍니다.

  • Service: Java

강의 평가 사이트

강의 평가 사이트는 회원 가입을 통해 각 학기별 강의 및 해당 강의 관련 교수님에게 강의를 평가하는 사이트 입니다.

사이트 화면의 기본 틀은 HTML/CSS와 Bootstrap을 사용하여 반응형으로 만들었습니다. 데이터 베이스를 통해 로그인과 회원가입, 강의평을 등록 할 수 있고 회원가입은 이메일 인증을 통하여 회원가입이 가능하게 만들었습니다. 세션을 통해 자신의 글 이외에는 삭제가 불가능하며 추천 버튼으로 강의 평가에 대해 추천이 가능합니다. 추천 버튼도 삭제와 마찬가지로 세션을 통해 한번 추천한 글을 다시 추천할 경우 추천이 최소되도록 하였습니다. 검색을 통하여 전체, 전공, 교양, 기타 및 최신순과 추천순으로도 검색이 가능하며 강의평은 등록하기 버튼으로 간단하게 모달창을 띄워 등록을 할 수 있으며 평가 내용에 상관이 없는 글 혹은 욕설인 경우는 신고하기 버튼으로 신고가 가능하며 신고를 할 경우 관리자의 메일로 신고가 되도록 하였습니다. 이메일 기능을 위해 외부 라이브러리인 activation.jar, mail.jar를 사용하였습니다.

  • Date: 2018.02 ~ 2018.03
  • Service: DB(MySQL), Java(Eclipse), JSP, HTML/CSS, bootstrap, apache

회원제 1:1 채팅 사이트

1:1 회원제 채팅 사이트는 간단한 회원 가입을 토해 각 회원들과 1:1로 채팅을 하고 가장 기본적인 자유게시판을 제작하였습니다.

사이트의 기본 틀은 HTML/CSS, Bootstrap을 이용하였고 페이지 이동은 Servlet을 통해 제작 하였습니다.

회원간의 1:1 채팅을 하면서 서로의 Profile을 확인 할 수 있도록 외부라이브러리인 cos.jar을 사용하여 파일을 업로드 할 수 있도록 하였습니다. 채팅을 하기 위해서는 로그인 또는 회원가입을 해야한 할 수 있으며 친구찾기를 통해 채팅방을 만들 수 있습니다. 채팅방은 회원이 재 접속 시에도 메시지 함에 남아 있으며 읽지 않은 메시지가 몇개인지 ajax를 이용하여 페이지를 새로고치지 않아도 확인 할 수 있도록 하였습니다. profile은 회원가입 이후에 회원관리 페이지에서 별도로 업로드 할 수 있으며 미등록시 기본의 익명 아이콘이 출력됩니다. 자유게시판은 파일을 업로드 및 다운로드가 가능하도록 제작하였고 채팅 및 게시판 내에서 Script구문을 사용할 수 없게 각 특수 기호들도 치환하였습니다.

  • Date: 2018.04 ~ 2018.05
  • Service: DB(MySQL), Java(Eclipse), JSP, HTML/CSS, bootstrap, apache, Servlet

영화 추천 사이트

영화 추천 사이트는 왓챠사이트를 기본 틀로 하여 각 영화를 회원들이 별점을 통하여 추천을 하고 장르와 별점으로 회원들에게 맞는 영화를 추천해주는 사이트 입니다.

사이트의 기본 틀은 HTML/CSS, Bootstrap을 이용하여 왓챠의 디자인을 참고로 하였습니다.
